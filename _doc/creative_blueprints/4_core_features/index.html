Core Capabilities

Four moving parts, one clean intake

Everything in scope today — a public form, a safety-critical triage gate, a live queue, and an automatic confirmation.

1

Patient intake form

Public, mobile-first, no login. Captures name, contact, reason for visit, and basic medical history — cleanly, with no clipboard to re-key.

!
Safety-critical

Urgent-symptom triage routing

Every submission is screened against an explicit, inspectable triage policy before the queue. Red-flag symptoms like chest pain route straight to nurse review — never the normal waiting line.

2

Live staff queue dashboard

Checked-in patients appear in real time with clear status — waiting → in progress → seen — plus a distinct nurse-review path for urgent cases. The desk's home screen.

3

Automatic SMS confirmation

The moment intake is received, the patient gets a text confirming it — quiet reassurance and cheap no-show insurance, sent without a single manual call.
